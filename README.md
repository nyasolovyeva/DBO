# Сравнение сервисов ДБО для самозанятых

Веб-приложение для сравнения банковских сервисов для самозанятых с использованием методов многокритериального анализа AHP и WSM.

## О проекте

Приложение позволяет пользователям:
- Сравнивать банковские сервисы ДБО по 16+ критериям
- Фильтровать сервисы по выбранным параметрам
- Получать взвешенные оценки (AHP и WSM рейтинги)
- Сравнивать до 6 сервисов в модальном окне
- Просматривать детальную информацию о каждом сервисе и банке

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| **Backend** | Python 3.13, Django 6.0 |
| **База данных** | PostgreSQL, ORM Django |
| **Математика** | NumPy (AHP, WSM)|
| **Frontend** | Bootstrap 5.3, JavaScript ES6 |

## Методы оценки

### AHP (Анализ иерархий)
- Метод Саати для расчета весов критериев
- Матрицы попарных сравнений для 6 основных групп критериев
- Проверка согласованности матриц (CR)

### WSM (Weighted Sum Model)
- Взвешенная сумма с использованием весов из AHP
- Нормализация не требуется

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourusername/DBO.git
cd DBO
```
### 2. Настройка базы данных
Нужно поменять поля USER и PASSWORD на свои учетные данные PostgresSQL
```
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'dbo_banks',
        'USER': 'postgres',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
### 3. Запуск сервера
- Для первого запуска, для сброса всех изменений в базе данных и создания нового суперпользователя нужно запустить файл "start and reset server.bat"
- Для обычного запуска, нужно запустить файл "start server.bat"
